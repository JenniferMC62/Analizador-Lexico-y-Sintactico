⚙️ Reglas de Gramática y Tokens

📋 Estructura del Programa

programa → lista_importaciones_opt metodo_principal lista_funciones_opt

📦 Importaciones

lista_importaciones_opt → lista_importaciones | ε
importacion → IMPORT ID PUNTOYCOMA

🎯 Método Principal

metodo_principal → VOID MAIN LPAR RPAR bloque

⚡ Funciones

declaracion_funcion → tipo_funcion FUNCTION ID LPAR parametros_opcional RPAR bloque
tipo_funcion → tipo_dato | VOID

💾 Tipos de Datos

tipo_dato → INT | FLOAT | STRING | BOOLEAN

📝 Sentencias

sentencia → declaracion_variable PUNTOYCOMA
sentencia → asignacion PUNTOYCOMA
sentencia → sentencia_if | sentencia_for | sentencia_while
sentencia_if → IF LPAR expresion RPAR bloque sentencia_if_prima
sentencia_while → WHILE LPAR expresion RPAR bloque
sentencia_for → FOR LPAR for_a PUNTOYCOMA expresion PUNTOYCOMA for_z RPAR bloque

🔢 Expresiones

expresion → expresion_logica
expresion_logica → expresion_and expresion_logica_prima
expresion_aditiva → expresion_multiplicativa expresion_aditiva_prima
expresion_primitiva → NUM_ENTERO | NUM_DECIMAL | CADENA | TRUE | FALSE | ID

🎫 Tokens Utilizados

Literales:

NUM_ENTERO NUM_DECIMAL CADENA TRUE FALSE ID

Operadores:

+ (SUMA) - (RESTA) * (MULT) / (DIV) % (MODULO) == (IGUAL) != (DIFERENTE) < (MENOR) > (MAYOR) <= (MENOR_IGUAL) >= (MAYOR_IGUAL) && (AND) || (OR) ! (NOT) = (ASIGNACION)

Delimitadores:

( (LPAR) ) (RPAR) { (LLLL) } (RRRR) [ (LCORCHETE) ] (RCORCHETE) ; (PUNTOYCOMA) , (COMA)

Palabras Reservadas:

INT FLOAT STRING BOOLEAN VOID IMPORT MAIN FUNCTION IF ELSE WHILE FOR RETURN PRINT TRUE FALSE CLASS TRY CATCH
